# Paper

This folder contains the final research manuscript and all supporting assets.

## Structure

```text
Paper/
├── main.tex
├── references.bib
├── sections/
│   ├── titlepage.tex
│   ├── abstract.tex
│   ├── introduction.tex
│   ├── literature_review.tex
│   ├── research_question.tex
│   ├── methods.tex
│   ├── results.tex
│   ├── discussion.tex
│   ├── conclusion.tex
│   └── recommendations.tex
├── figures/
├── tables/
└── appendix/
```

## Notes

- `main.tex` is the primary entry point for the paper.
- `references.bib` stores bibliography entries.
- `sections/` contains the manuscript chapters split by topic.
- `figures/`, `tables/`, and `appendix/` store supporting material.
